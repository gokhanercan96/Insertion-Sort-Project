# Insertion Sort Project 

## 1) [22,27,16,2,18,6] dizininin Insert Sort türüne göre aşamaları

```
[22,27,16,2,18,6] n
[2,6,16,22,18,27] n-1
[2,6,16,22,18,27] n-2
[2,6,16,18,22,27] 1
```

## 2) Big O Notation

```
O(n²)
```
